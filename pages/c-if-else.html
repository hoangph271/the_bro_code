Đầu tiên là cái này, toán tử 
"%"
có nghĩa là lấy số dư của phép chia, nhá...?
Ví dụ, cái đoạn này: 
int a = 15 % 4;
Thằng a sẽ có giá trị là 3 nhá...?
Còn nếu là cái đoạn này: 
int b = 8 % 4;
Thằng b sẽ có giá trị là 0, OK...?
Tiếp nha, đọc thử code dưới này...?
#include

int main() {
  int num = 0;
  printf("Enter the number");
  scanf("%d", &num);

  if (num % 2 == 0) {
    printf("%d is even number", num);
  }

  return;
}
Thì chỉ có đoạn này là mới thôi đúng không...? 
if (num % 2 == 0) {
  printf("%d is even number", num);
}
Đoạn mới này gọi là cấu trúc điều kiện, cấu trúc điều kiện có dạng như ri: 
if (dieu_kien) {
  // Lam chi do
}
Ý nghĩa là, nếu đoạn code trong 
dieu_kien
mà "đúng", thì đoạn code 
// Lam chi do
sẽ được thực thi, còn không thì sẽ bỏ qua đoạn code 
// Lam chi do
...!
Thấy chữ "đúng" anh để trong ngoặc ha, "đúng" và "sai" là kết quả của những phép toán như này: 
So sánh bằng: ==
So sánh khác: !=
So sánh nhỏ hơn: <
So sánh lớn hơn: >
So sánh nhỏ hơn hoặc bằng: <=
So sánh lớn hơn hoặc bằng: >=
Ví dụ: 
5 == 3 cho kết quả sai
2 != 0 cho kết quả đúng
2 >= 0 cho kết quả đúng
2 > 2 cho kết quả sai
2 < 0 cho kết quả sai
2 <= 2 cho kết quả đúng
Rứa thì, quay lại đoạn code cũ...! 
if (num % 2 == 0) {
  printf("%d is even number", num);
}
Đoạn này sẽ in ra thông báo num là số chẵn (even) nếu "kết quả phép chia num cho 2" bằng với 0...!
Hiểu chưa...?
